# vector-embedding-demo
Simple document embedding pipeline using SentenceTransformers

This project demonstrates how to convert text documents into vector embeddings using the SentenceTransformer model.

Steps:
1. Load text documents.
2. Generate embeddings using all-MiniLM-L6-v2
3. Store vectors for semantic search pipelines.

Tech stack:
Python, SentenceTransformers, NumPy
